Ce que je vous propose

Déclaration préalable

Maximum 170 m² de surface de plancher.
Je peux réaliser les déclarations préalables de travaux pour vos projets:

Permis de construire

Je vous propose de réaliser votre dossier de demande de permis de construire sur la base de croquis ou d’esquisses réalisés par vos soins.
Toutefois, je contrôle si votre projet est adaptable sur votre terrain (par rapport aux règles d’urbanisme notamment). Pour ceux qui ont déjà fait l’avant-projet avec moi, une remise sera faite pour le dossier permis de construire.

Volet paysager

Vous avez réalisé vous-même les plans, mais vous n’arrivez pas à faire le document graphique d’insertion sur le site, qui est obligatoire pour le dépôt du permis de construire, ou de la déclaration préalable de travaux (pièce PCMI 6 ou DP 6).

Je vous propose de réaliser le volet paysager afin de mettre toutes les chances de votre côté pour que votre permis soit accepté (cette pièce étant très importante).

Image de synhtèse 3D

Vous n’arrivez pas à vous projeter dans votre projet ou vous n’êtes pas décidé sur les couleurs.

Je vous propose de réaliser une image de synthèse 3D (voir portfolio) qui vous permettra de visualiser vos futurs espaces.
